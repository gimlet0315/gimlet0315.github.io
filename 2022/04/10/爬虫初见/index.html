发表于 2022-04-10 | 分类于 python

等着改论文，闲来无聊学一下爬虫。因为对数据分析还挺感兴趣，先用快乐老家当对象训练一下。
之前接触过，python数据处理主要就是爬虫，当时练习的是疫情数据和一个大学的什么；后来用户行为与数据分析，前期是一些理论和概念，后面就全是技术方面的了。刚好捡起来。

scrapy：

首先当然是scrapy的介绍，按照菜鸟Scrapy 入门教程 | 菜鸟教程 (runoob.com)给的：

Scrapy 是用 Python 实现的一个为了爬取网站数据、提取结构性数据而编写的应用框架。
Scrapy 常应用在包括数据挖掘，信息处理或存储历史数据等一系列的程序中。
通常我们可以很简单的通过 Scrapy 框架实现一个爬虫，抓取指定网站的内容或图片。

scrapy：给的定义落在“应用框架”。因为之前接触的都是“安装包”，所以搜了一下：Python模块、包、库、框架_studyer_domi的博客-CSDN博客_python的框架和库（没看懂……）。

from scrapy.spiders import Spider

class BlogSpider(Spider):
    name = 'woodenrobot'
    start_urls = ['http://woodenrobot.me']

    def parse(self, response):
        titles = response.xpath('//a[@class="post-title-link"]/text()').extract()
        for title in titles:
            print title.strip()
