Begin!

发表于 2022-03-16

今天遇到了很多bug，稍微记录一下
首先就是hexo init报错，Cannot find module ‘hexo’ from xxx
后来先是更改了管理员权限，把nodejs权限改成users：完全控制
然后又用npm cache verify, npm cache clean –force
npm cache clean –force之后成功了。

不过插一下前情：
本来我的电脑是固态硬盘，而不是跟的教程里的分C、D盘。所以在更改环境变量的时候出了问题
1，新建node_cache、node_global文件夹之后，改系统变量。不过我当时node_global文件夹下还未生成node_modules子文件夹
2，更改用户变量后，明明已经把Roaming下npm变量删掉了，结果后来重启重新查看的时候发现不知道什么时候又出来了
3，因为2，导致我很多包比如webpack、hexo全都按到了Roaming/npm下，后来又重复步骤很多次
4，接着就是开头的问题，在blog文件夹下安装hexo，一直安不了。

总之，虽然不清楚到底是权限还是变量没配好的原因，总算是又继续朝后推进了一点点

后来就比较顺利了。稍微说一下部署next主题时遇到的问题
next默认四个主题：Muse、 Mist、Pisces、 Gemini，其中有一些设置只是only for（Muse、 Mist）。我折腾主题的时候，可能因为改了一些显示，比如时间线和目录之类的，直接导致网页打开白板。我非常讨厌bug，加上又是在git bush里下的命令，所以没办法，只能慢慢找。不过git bush还是非常人性化的，直接给我标出了文件的哪一行，config.yml注释也很清晰，所以最后还是比较顺利。
